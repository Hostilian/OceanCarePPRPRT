1

Register for Cleanup Event

User: Alex (The Volunteer)

Step 1: Home Screen

User starts on home screen and selects "Find Events"
User Actions:
  1. User is presented with the Home Screen
  2. User taps the "Find Events" button (highlighted in blue)
  3. System transitions to Events listing screen

Step 2: Events & Confirmation Pop-up

Facilitator places confirmation pop-up (paper cutout) on top of events screen
User Actions:
  1. User sees Cleanup Events Screen with available events
  2. User taps "Join" button for "Ocean Beach Cleanup"
  3. Facilitator places Confirmation Pop-up on top (separate paper cutout)
  4. User taps "Confirm" to complete registration
  5. Pop-up is removed, showing "Registration Successful" message
2

Report Marine Debris

User: Maria (The Concerned Citizen)

Step 1: Blank Report Form

User navigates to Report Issue screen with blank form
User Actions:
  1. User starts on the Home Screen
  2. User taps "Report Issue" button
  3. System presents blank Report Form with fields for:
    • Photo/Image upload area
    • Location (GPS coordinates)
    • Description text area

Step 2: Completed Report & Submission

User fills form with debris photo sketch, location, and description
User Actions:
  1. User "taps" the photo area to add image (facilitator draws simple sketch)
  2. User enters location data (e.g., by writing on a post-it or verbally stating coordinates)
  3. User adds description of debris (plastic bags, bottles, etc.)
  4. User taps "Submit Report" button
  5. System shows "Report Submitted" confirmation message
  6. Report data is stored for review by conservation team
3

Make a Donation

User: John (The Donor)

Step 1: Donate Screen - Amount Selection

User selects $20 donation amount from available options
User Actions:
  1. User starts on the Home Screen
  2. User taps the "Donate" button
  3. System presents Donate Screen showing:
    • Current campaign: "Save the Turtles"
    • Multiple donation amounts: $5, $10, $20, $50
  4. User taps "$20" button (highlighted in green)

Step 2: Confirmation Pop-up

Facilitator places thank you pop-up (paper cutout) confirming donation
User Actions:
  1. User taps "Donate Now" button to process donation
  2. Facilitator places "Thank You" Pop-up on top (separate paper cutout)
  3. Pop-up displays:
    • Confirmation message: "Your $20 donation has been received!"
    • Impact statement: "Saves 5 sea turtles from plastic"
    • "Done" button to close
  4. User taps "Done" to close pop-up
  5. Pop-up is removed, returning to donate screen or home screen

üìã Paper Prototype Method Summary

This paper prototype demonstrates an interactive method for user testing without requiring functional code. Key advantages:

  • Low-Fidelity: Hand-drawn style allows quick iterations based on user feedback
  • Interactive Pop-ups: Separate paper cutouts simulate dialog boxes and transitions
  • User-Centered Design: Focuses on three realistic user personas and their core tasks
  • Complete Flows: Each task is complete from start to finish with clear outcomes
  • Facilitator-Driven: Allows for flexible interaction and verbal explanations during user testing
  • Cost-Effective: Requires only paper, pen, and user participants

How to Use This Prototype for User Testing

  1. Print all 6 images on paper or cardstock
  2. Cut out the pop-up elements (images 2 and 6) from their backgrounds
  3. Prepare printed copies of the main screens (images 1, 3, and 5)
  4. During testing, facilitate by placing pop-up cutouts on top of main screens
  5. Record user feedback and note any confusion or improvement suggestions
  6. Iterate design based on feedback before moving to digital prototype
